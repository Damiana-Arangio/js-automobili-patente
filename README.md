<h1 align="center">Snack – Automobili e patente</h1>

Piccoli esercizi JavaScript che lavorano con array di oggetti usando la sintassi ES6 e l’output in console.

## Obiettivi

### 1. Automobili per alimentazione

- Creare un array di 10 oggetti `automobile` con proprietà:
  - `marca`
  - `modello`
  - `alimentazione` (benzina, diesel, gpl, elettrico, metano, …)
- Suddividere l’elenco in tre array distinti:
  - auto a **benzina**;
  - auto a **diesel**;
  - auto con **altre alimentazioni**.
- Stampare separatamente in console i tre insiemi di veicoli.

### 2. Persone e idoneità alla guida

- Creare un array di oggetti `persona` con proprietà:
  - `nome`
  - `cognome`
  - `eta`
- Generare, tramite `map`, un nuovo array di stringhe che per ogni persona indichi:
  - nome e cognome;
  - se **può guidare** (età ≥ 18 anni) oppure **non può guidare**.
- Stampare il nuovo array di frasi in console.

## Tecnologie utilizzate

- JavaScript
- HTML5 (di base per collegare lo script)
